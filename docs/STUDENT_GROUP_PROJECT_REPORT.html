1. Problem Statement and Objectives

Skin disease diagnosis from images can improve accessibility and preliminary screening support. The objective is to build a practical image classifier that predicts one of eight disease categories from dermoscopic or clinical images.

Target Classes

  • Acne
  • Basal Cell Carcinoma (BCC)
  • Benign Keratosis-like Lesions (BKL)
  • Atopic Dermatitis (Eczema)
  • Actinic Keratosis (AK)
  • Melanoma
  • Psoriasis
  • Tinea (Ringworm)

Project Objectives

  • Build a robust and CPU-friendly ViT pipeline.
  • Improve accuracy over baseline single-model approaches.
  • Integrate the best model into the web application runtime.
  • Generate reproducible evaluation artifacts (metrics and confusion matrix).
  • Document outcomes with visual evidence and class-wise analysis.

Since some dataset versions share source images, overlap-aware checks were performed. A leakage-safe subset of the test split (excluding overlap with hf_hybrid_220 train and validation) still showed improved accuracy, supporting generalization of the final ensemble.

3. Model Architecture and Training Pipeline

Core Approach

  • Primary architecture: vit_tiny_patch16_224 variants.
  • Model-specific image sizes: 160 and 224.
  • Training regularization: label smoothing, drop-path, random erasing, TTA in selected runs.
  • Optimization under CPU budget with early stopping and curated profile runs.
  • Final runtime uses weighted ensemble inference.

5. Class-wise Performance Analysis

Observations

  • Acne and BCC classes show very strong recall and stable precision.
  • AK and Melanoma classes maintain balanced precision-recall behavior.
  • Eczema remains the most challenging class in recall (0.6087), indicating higher confusion with related classes.
  • Tinea shows high precision (0.8947) with moderate recall (0.7391), suggesting conservative prediction behavior.

6. Confusion Matrix and Error Patterns

Key Error Modes

  • Eczema misclassified mostly as Psoriasis and Acne in selected instances.
  • AK occasionally predicted as BCC.
  • Tinea occasionally confused with Psoriasis and Acne.
  • Acne has perfect recall in the final run (23 out of 23).

These patterns are clinically reasonable where visual similarity is high. Future work should target difficult class boundaries with targeted augmentation and class-specific data curation.

8. Limitations and Risk Assessment

Current Limitations

  • Accuracy is strong for a CPU-constrained student project, but still below 90% target.
  • Certain classes with visual overlap (Eczema, Psoriasis, Tinea) remain harder.
  • Dataset overlap risk across versions requires strict split governance.
  • No uncertainty calibration or confidence rejection policy yet.

Risk Mitigation Already Applied

  • Leakage-safe subset verification included before final claims.
  • Ensemble-based stabilization rather than single checkpoint dependence.
  • Consistent metric export and reproducible evaluation scripts.

Ethical Statement

This system is for educational and research use only and does not replace clinical diagnosis by qualified medical professionals.

9. Conclusion and Future Scope

Conclusion

The project successfully delivered an integrated ViT-based skin disease classifier with a final ensemble score of 85.33% accuracy and 86.10% macro precision on the primary test split. The workflow covers model training, optimization, deployment integration, and reporting with reproducible artifacts.

Future Scope

  • Train higher-capacity ViT variants with longer schedules on curated non-overlapping data.
  • Add confidence calibration and abstention logic for uncertain samples.
  • Include external benchmark validation to estimate out-of-distribution robustness.
  • Add clinician-in-the-loop review module in the web application.
